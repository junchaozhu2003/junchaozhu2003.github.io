Nanchang City Symbols

Visual Identity System / Cultural Communication / Print Design

A visual communication project exploring how simplified city symbols can introduce Nanchang to international audiences through postcards, stamps, posters, stickers, calendars and print-based visual storytelling.

Project Type Visual Communication Graduation Project
Focus City Symbols / Tourism Communication / Print Design
Role Visual identity design, icon development, layout design, print applications
Tools Photoshop / InDesign / Illustrator / Creative Production
Brand Identity Visual Communication City Branding Print Design

Section 01

Context & Challenge

Nanchang is less internationally recognised than major tourist cities in China, so the project explores how a focused set of landmark symbols can create an accessible first point of recognition for visitors. I developed the identity around three city references: Tengwang Pavilion for history, Star Ferris Wheel for modern life and Jiangxi Art Museum for culture. The challenge was to make these references clear, visually consistent and usable across tourism-focused print outcomes.

Section 02

Research Direction

I used city branding, tourist communication and wayfinding references to define a clearer visual direction. The research helped me focus on landmark selection, language-independent symbols and print-based information hierarchy. Instead of relying on long written explanations, the project uses simplified forms that can support quick recognition across postcards, posters and visitor-facing materials.

Section 03

Icon Development

The icon development focused on selecting landmarks, simplifying architectural details and testing whether the forms stayed recognisable at smaller scales. I refined proportions, line weight and the relationship between symbol and type so the icons could operate as a consistent visual identity system rather than separate illustrations.

Section 04

Visual System

I developed the system around symbol consistency, colour palette testing and light/dark application logic. The layout approach keeps the identity flexible enough for posters, calendars, stamps and object mockups while maintaining a recognisable visual language across different formats.

Section 05

Applications

I tested the system across calendar layouts, clothing mockups, an apron mockup and combined postcard, stamp and souvenir-style applications. The application tests helped check whether the identity could extend beyond a single logo and remain clear across both information-led and promotional formats.

Section 06

Final Tourism Poster

Final tourism poster combining landmark symbols, ranking information, transport icons and bilingual communication to introduce Nanchang to international visitors.

Section 07

Print & Product Applications

A set of print-based applications including calendars, stamps, clothing and souvenir-style objects, showing how the visual identity system can extend across different formats.

Reflection

This project helped me understand how visual identity can support cultural communication, while also showing the risk of oversimplifying a city into a limited set of symbols. The final system aims to create an accessible entry point rather than a complete representation of Nanchang, balancing cultural identity, visual clarity and practical application across different printed formats.
